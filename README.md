# Unburden

A minimalist iOS journaling app to release worries, record gratitude, and track mood. All data stored privately in your iCloud.

## Features

- **Worry Box**: Write down worries and watch them fade away
- **Gratitude Box**: Keep a running list of things you're grateful for
- **Mood Tracker**: Log your mood with a simple emoji picker

## Requirements

- iOS 17.0+
- iCloud account (for data sync)

## Tech Stack

- SwiftUI
- MVVM architecture with `@Observable`
- iCloud Documents for storage

## Privacy

All data is stored exclusively in your personal iCloud account. No analytics, no tracking, no servers.

[Privacy Policy](https://gist.github.com/minac/27997e9fa5f779a31d2ada87db2efcb6)

## Development

Open `Unburden.xcodeproj` in Xcode and build for iOS Simulator or device.
